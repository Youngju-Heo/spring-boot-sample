# Spring boot 에서 servelet engine을 이용한 MVC 구현 샘플

## 주요 스펙

* build 사양
  * JDK 11+
  * Spring Boot 2.7.4.RELEASE
  * 빌드도구: gradle
  
* 주요 적용항목
  * HikariCP
  * MyBatis
  * log4j2
  * i18n
  * PostgreSQL

* 기타
  * spring-boot-java-web
    * JAVA 언어 기반 구현 예제
	* lombok annotation 도구 활용
  * spring-boot-kotlin-web
    * Kotlin 언어 기반 구현 예제
